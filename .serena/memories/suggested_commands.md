# Suggested Commands

For developing in this VitePress project, use the following commands (assuming npm or yarn):

- `npm run dev` or `yarn dev`: Start the local development server (runs `vitepress dev docs`).
- `npm run build` or `yarn build`: Build the static site for production (runs `vitepress build docs`).
- `npm run serve` or `yarn serve`: Preview the built production site locally (runs `vitepress serve docs`).
